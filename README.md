                                   # Cat Cloud

Ask a cat a yes-or-no question and the dark answers back.

## About

Cat Cloud is an interactive oracle. You type a question that can be answered
with YES or NO, a ripple of sonar spreads across a dark pool, and the Cat Cloud
returns a verdict — along with a quiet, slightly unsettling reading of the
person who asked.

It is part toy, part mirror. The answer is only half the point; the rest is what
the question says about you.

## What it does

- Type any yes/no question on the home screen.
- A water-ripple animation plays while the cat "listens." The number of ripples
  is drawn from your question.
- The result screen shows:
  - a number and a category word (FEAR, AROUSAL, TERRITORY, and so on),
  - a short passage about the cat and that category,
  - a personal reading of you, drawn from the 48 Laws of Power,
  - and the final answer: **YES** or **NO**.
- Every result is derived from the question itself, so the same question always
  gives the same answer.

## Run it

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
